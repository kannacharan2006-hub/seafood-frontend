## ⚡ About

Seafood ERP is a next-generation business management platform designed for modern seafood operations.

Built with performance, scalability, and simplicity in mind, the platform helps businesses manage workflows digitally through a clean and responsive experience.

Whether it’s inventory tracking, order handling, billing, or operational management — Seafood ERP is focused on making seafood businesses faster and smarter.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/kannacharan2006-hub/seafood-frontend.git
```

## Navigate to Project

```bash
cd seafood-frontend
```

## Install Dependencies

```bash
flutter pub get
```

## Run Application

```bash
flutter run
```
